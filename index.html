UNIT I: Sets, Logic and Relations

1. Sets

Definition: A set is a well-defined collection of distinct objects, called elements.

Example: A = {1, 2, 3}

Subset: A set A is a subset of B if every element of A is also an element of B. Symbolically, A ⊆ B.

Power Set: The power set of a set A is the set of all possible subsets of A. If A has n elements, then P(A) has 2ⁿ elements.

Operations on Sets:

  • Union: A ∪ B = {x | x ∈ A or x ∈ B}
  • Intersection: A ∩ B = {x | x ∈ A and x ∈ B}
  • Difference: A − B = {x | x ∈ A and x ∉ B}

2. Propositional Logic

Proposition: A proposition is a statement that is either true (T) or false (F), but not both.

Logical Connectives:

  • ¬p : Negation (NOT)
  • p ∧ q : Conjunction (AND)
  • p ∨ q : Disjunction (OR)
  • p → q : Implication
  • p ↔ q : Biconditional

Truth Table (Implication):

p   q   p → q
T   T     T
T   F     F
F   T     T
F   F     T

Rules of Inference:

  • Modus Ponens: If p → q and p is true, then q is true.
  • Modus Tollens: If p → q and q is false, then p is false.

3. Predicate Logic

Predicate: A predicate is a statement containing variables, which becomes a proposition when values are assigned.

Example: P(x): x > 5

Quantifiers:

  • Universal Quantifier (∀): For all
  • Existential Quantifier (∃): There exists

Validity: An argument is valid if the conclusion logically follows from the premises.

Normal Forms:

  • CNF: Conjunction of disjunctions
  • DNF: Disjunction of conjunctions

4. Proof Techniques

Direct Proof: Assume the hypothesis is true and logically derive the conclusion.

Proof by Contraposition: To prove p → q, prove ¬q → ¬p.

Proof by Contradiction: Assume the statement is false and reach a contradiction.

5. Counting Principles

Principle of Inclusion and Exclusion:

n(A ∪ B) = n(A) + n(B) − n(A ∩ B)

Pigeonhole Principle: If n+1 objects are placed into n boxes, at least one box contains more than one object.

Permutation:

nPr = n! / (n − r)!

Combination:

nCr = n! / [r!(n − r)!]

6. Mathematical Induction

Steps:

  • Base Case
  • Induction Hypothesis
  • Inductive Step

Principle of Well Ordering: Every non-empty set of positive integers contains a least element.

7. Relations

Binary Relation: A relation R from A to B is a subset of A × B.

Properties:

  • Reflexive: (a, a) ∈ R
  • Symmetric: (a, b) ∈ R ⇒ (b, a) ∈ R
  • Transitive: (a, b) and (b, c) ⇒ (a, c)

Equivalence Relation: A relation that is reflexive, symmetric, and transitive.

Equivalence Class: [a] = {x | (a, x) ∈ R}

UNIT II: Functions, Order Relations and Boolean Algebra

1. Functions

Definition: A function f from set A to set B is a relation that assigns exactly one element of B to each element of A.

Notation: f : A → B

Types of Functions:

  • One-to-One (Injective): Different inputs have different outputs.
  • Onto (Surjective): Every element of codomain has a pre-image.
  • Bijective: Both one-to-one and onto.

Growth of Functions:

  • Constant: O(1)
  • Logarithmic: O(log n)
  • Linear: O(n)
  • Quadratic: O(n²)
  • Exponential: O(2ⁿ)

Permutation Function: A bijective function from a set onto itself.

2. Order Relations

Partial Order Relation: A relation R on a set A is called a partial order if it is:

  • Reflexive
  • Antisymmetric
  • Transitive

The pair (A, R) is called a Partially Ordered Set (POSET).

Lattice: A POSET in which every pair of elements has a least upper bound (join) and a greatest lower bound (meet).

Join: a ∨ b
Meet: a ∧ b

3. Boolean Algebra

Definition: Boolean algebra is an algebraic structure defined on a set B with two binary operations (+, ·), a unary operation (′), and two distinct elements 0 and 1.

Basic Boolean Laws:

  • Identity Law
  • Domination Law
  • Idempotent Law
  • Complement Law
  • Commutative Law
  • Associative Law
  • Distributive Law
  • De Morgan’s Laws

Minimization of Boolean Expressions: Reducing a Boolean expression to its simplest form using Boolean laws or Karnaugh Maps (K-map).

4. Number Theory

Greatest Common Divisor (GCD): The largest positive integer that divides two or more integers.

Least Common Multiple (LCM): The smallest positive integer divisible by given integers.

Relation between GCD and LCM:

GCD(a, b) × LCM(a, b) = a × b

Prime Number: A number greater than 1 that has exactly two distinct positive divisors: 1 and itself.

5. Recurrence Relations

Definition: A recurrence relation is an equation that defines a sequence using previous terms.

Linear Recurrence Relation:

aₙ = c₁aₙ₋₁ + c₂aₙ₋₂ + … + cₖaₙ₋ₖ

First Order Linear Recurrence Relation:

aₙ = c aₙ₋₁ + d

Generating Function: A power series used to represent a sequence.

Divide and Conquer Recurrence:

T(n) = aT(n/b) + f(n)

Master’s Theorem:

  • If f(n) = O(nlogba − ε) → T(n) = Θ(nlogba)
  • If f(n) = Θ(nlogba) → T(n) = Θ(nlogba log n)
  • If f(n) = Ω(nlogba + ε) → T(n) = Θ(f(n))

UNIT III: Group Theory

1. Semi-Group

Definition: A non-empty set S together with a binary operation (*) is called a semigroup if it satisfies the associative property.

Associative Property: (a * b) * c = a * (b * c), for all a, b, c ∈ S.

Example: (N, +) is a semigroup.

2. Monoid

Definition: A monoid is a semigroup with an identity element.

Identity Element: An element e ∈ S such that a * e = e * a = a, for all a ∈ S.

Example: (N, +) with identity 0 is a monoid.

3. Group

Definition: A group is a non-empty set G with a binary operation (*) satisfying:

  • Closure
  • Associativity
  • Identity element
  • Inverse of every element

Example: (Z, +) is a group.

4. Identity and Inverse (Uniqueness)

Uniqueness of Identity: A group has only one identity element.

Uniqueness of Inverse: Every element in a group has exactly one inverse.

5. Homomorphism and Isomorphism

Homomorphism: A function f : G → G′ is a homomorphism if f(a * b) = f(a) * f(b).

Isomorphism: A bijective homomorphism is called an isomorphism.

If two groups are isomorphic, they have the same algebraic structure.

6. Subgroup

Definition: A non-empty subset H of a group G is called a subgroup if H itself forms a group under the operation of G.

Subgroup Test:

  • Closure under operation
  • Closure under inverse

Example: (2Z, +) is a subgroup of (Z, +).

7. Cosets and Lagrange’s Theorem

Left Coset: aH = {ah | h ∈ H}

Right Coset: Ha = {ha | h ∈ H}

Lagrange’s Theorem: If H is a subgroup of a finite group G, then the order of H divides the order of G.

|G| = |H| × number of cosets

8. Permutation Group

Permutation: A bijection from a set onto itself.

Permutation Group: The set of all permutations of a set forms a group under composition.

9. Cayley’s Theorem

Statement: Every group is isomorphic to a subgroup of a permutation group.

Note: Proof is not required.

10. Normal Subgroup and Quotient Group

Normal Subgroup: A subgroup N of G is normal if gN = Ng for all g ∈ G.

Quotient Group: The set of cosets of a normal subgroup forms a group, denoted by G/N.

11. Groups and Coding

Group theory is used in coding theory for error detection and correction, such as cyclic codes and linear codes.

UNIT IV: Graph Theory

1. Graph Terminology

Graph: A graph G is an ordered pair (V, E), where V is a set of vertices and E is a set of edges.

Vertex (Node): A point in a graph.

Edge: A connection between two vertices.

Degree of a Vertex: Number of edges incident on the vertex.

Path: A sequence of vertices connected by edges.

Cycle: A path that starts and ends at the same vertex.

2. Planar Graph

Definition: A graph is planar if it can be drawn in a plane without any edges crossing.

Regions (Faces): Areas bounded by edges in a planar graph.

3. Euler’s Formula

Statement: For a connected planar graph:

V − E + F = 2

Where V = number of vertices, E = number of edges, F = number of faces.

Note: Proof is required in exams.

4. Euler Path and Euler Circuit

Euler Path: A path that uses every edge of a graph exactly once.

Euler Circuit: An Euler path that starts and ends at the same vertex.

Condition:

  • Euler Circuit → All vertices have even degree
  • Euler Path → Exactly two vertices have odd degree

5. Hamiltonian Path and Circuit

Hamiltonian Path: A path that visits each vertex exactly once.

Hamiltonian Circuit: A Hamiltonian path that starts and ends at the same vertex.

Note: No general necessary and sufficient condition exists.

6. Chromatic Number

Chromatic Number (χ): Minimum number of colors required to color the vertices of a graph such that no two adjacent vertices have the same color.

7. Five Color Theorem

Statement: Every planar graph can be colored using at most five colors.

Note: Proof is required in exams.

8. Shortest Path Algorithm

Dijkstra’s Algorithm: Used to find the shortest path from a source vertex to all other vertices in a weighted graph.

9. Minimal Spanning Tree

Spanning Tree: A subgraph that includes all vertices of the graph and is a tree.

Minimum Spanning Tree (MST): A spanning tree with minimum total edge weight.

Algorithms:

  • Prim’s Algorithm
  • Kruskal’s Algorithm

10. Graph Traversal

Depth First Search (DFS): Explores as far as possible before backtracking.

Breadth First Search (BFS): Explores all neighbors of a vertex before moving deeper.

11. Trees from DFS and BFS

DFS and BFS generate spanning trees called DFS tree and BFS tree respectively.

12. Connected Components

A connected component is a maximal set of vertices such that there is a path between every pair of vertices.

13. Graph Complexity Analysis

Time Complexity:

  • DFS: O(V + E)
  • BFS: O(V + E)
